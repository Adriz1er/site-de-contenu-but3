On parle souvent de WLAN.
Toutes les normes [^1] sont des extensions de IEEE 802.11
Wifi ne vient pas de Wireless Fidelity
## filaire vs wifi

|                | filaire                         | wifi                                | qui gagne |
| -------------- | ------------------------------- | ----------------------------------- | --------- |
| installation   | coûts et difficultés de câblage | coûts et difficultés de câblage     | égalité   |
| mobilité       |                                 | wifi est plus pratique évidemment   | wifi      |
| fiabilité      | beaucoup moins de perturbations |                                     | filaire   |
| bande passante | reste meilleur                  | des progrès notables mais aléatoire | filaire   |
| sécurité       | des menaces                     | encore + de menaces                 | filaire ! |
